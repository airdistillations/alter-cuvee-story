# ALTER Cuvée — Investor one-pager

Password-gated static site deployed on GitHub Pages.
**URL:** _set after GitHub Pages is enabled_
**Password:** `altermethode`

## Files

| File | Purpose |
|------|---------|
| `alter-cuvee-pitch.html` | Editable source. **Never the deployed version.** |
| `index.html` | Encrypted deployed version. Re-generated by `encrypt.sh`. |
| `staticrypt-template.html` | ALTER-themed password prompt. |
| `.staticrypt.json` | AES salt (keep committed — stable across re-encrypts). |
| `encrypt.sh` | Helper: copies source → `index.html` + runs staticrypt. |
| `pitch-assets/` | Images + logo SVGs. |

## Edit & publish workflow

```bash
# 1. Edit the source
open alter-cuvee-pitch.html   # or any editor

# 2. Re-encrypt
./encrypt.sh

# 3. Commit and push — GitHub Pages redeploys automatically
git add .
git commit -m "update deck copy"
git push
```

## Changing the password

Edit `PASSWORD="..."` in `encrypt.sh`, then run `./encrypt.sh`. Old links will stop working.

## Privacy

- The repo is **private**. Keep it that way — `alter-cuvee-pitch.html` contains the plaintext deck.
- The *deployed* site is public (anyone with the URL). Only `index.html` + assets are served.
- StaticCrypt does real AES-256 encryption in the browser — View Source on the deployed page shows ciphertext.

## Notes

- GitHub Pages free tier: 1 GB repo, 100 GB/month bandwidth — far above what this deck needs.
- Deploy time after push: ~30–60 seconds.
